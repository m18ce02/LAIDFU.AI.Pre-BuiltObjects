## 版本与更改日志
- **v1.0.0** (2025-12-05): 初始版本，实现自动化订单处理功能的采购订单 AI 助手。

## 功能概述
- 本示例展示了一款 采购 AI 助手，支持简体中文、繁体中文与英文 三语交互，可在【AI广场】与 aiM18 App 中通过对话创建与查询采购订单，提升采购流程的智能化与自动化水平。
核心功能：
1. **多语言交互**：提供3种语言版本的AI助理，满足跨地区采购团队的使用需求。
2. **意图识别与单据生成**：自动识别“创建采购订单”意图，并调用模块 [AI 对话转采购订单数据载体] 生成单据并提交审批。
3. **智能订单创建**：支持基于 PQ 生成 PO  或 根据用户提供的采购明细创建 PO。
4. **自动化流程校验**：自动确认供应商ID与基础资料，关键字段完整性检查，产品单价多来源获取（报价单、采购历史记录、价格表），内置业务规则与金额风险检查，确保数据合规。
5. **ERP 自动对接**：生成并同步 aiM18ERP 采购订单，实现对话到 ERP 的无缝集成。
6. **结果通知**：任务完成后推送处理结果与快捷访问链接。
7. **跨平台支持**：支持移动端 App，实现随时随地执行采购任务。
 
## 产品功能要求
使用本Objects至少需要购买 aiM18 并安装以下模块功能：
- 智筑引擎LAIDFU (AI Builder)
- 万达宝智能广场 (AI Square)
- 聊天行动机器人(Chat-to-Action Robot)
- 供应链管理 (Supply Chain Management)

## 安装与配置步骤
1. 从本文件夹根目录下载安装包压缩文件“install_file.zip”。
2. 将压缩文件解压至本地文件夹。
3. 参考解压后的“Objects安装与配置文档_AI 对话转采购订单数据载体_v1.0.0.pdf”完成配置。
4. 在 aiM18 中导入 Objects 文件，验证功能是否正常运行。
